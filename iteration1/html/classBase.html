Detailed Description

Class representing a immobile base within the Arena.

Base can enhance a Robot. If a Robot touches the Base, it becomes a super robot.

Base have the capability of updating their own position when asked, and also track their own velocity and heading. They have a touch sensor for responding to collision events which is activated/deactivated on collision events.
